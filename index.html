The project

Paradar is a web application that helps National Crisis Center tracking vital and/or vital assets travelling across Belgium. Every asset is tracked via a physical tracker or using a browser from a smartphone or a laptop.

This project also helps for crises that should be handled on a national level.

Want to know how this grid works, what the headers look like and what kind of buttons we have? Go to the styleguide. Before you ship, don't forget to remove "styleguide-remove-me.css", and replace all the favicons.
