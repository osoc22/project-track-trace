The project

Paradar is a web application that helps National Crisis Center tracking persons and/or vital assets travelling across Belgium. Every asset is tracked via a physical tracker or by using a smartphone or a laptop.

This project is a tool that can be used during national crises

By using the power of modern browser technologies and Internet Of Things (or IoT), we aim to make your life a little easier in times of chaos.
